## Movie Recommendation System
This project is a machine learning-based movie recommendation system that suggests similar movies based on user preferences. It uses cosine similarity to find relationships between movies and provides an interactive web interface built with Streamlit.

## Features
* Recommend similar movies instantly
* Clean and interactive UI using Streamlit
* Movie posters fetched using TMDB API
* Content-based filtering approach

## Tech Stack
* Python
* Pandas, NumPy
* Scikit-learn
* Streamlit
* TMDB API

## How it Works
The system processes movie metadata and creates similarity scores using vectorization techniques. When a user selects a movie, the system recommends top similar movies based on cosine similarity.

## Deployment
This project can be deployed using Streamlit Cloud for easy access and sharing.
